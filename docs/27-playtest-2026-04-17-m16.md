# 마일스톤 16 배치 검증 로그

## 목적

장비 조합 1차 구현이 실제 플레이 흐름에서 끝까지 동작하는지 검증한다.

검증 대상:

- `Basic Box` 중복 획득
- 같은 장비 3개 누적
- `같은 장비 3개 -> Lv +1` 조합
- 장착 장비 자동 재장착
- 강화 수치 재적용

## 검증 방식

- 원본 Unity Editor는 유지했다.
- `unity/TinyFactoryPrototype/` 복제본 `unity/TinyFactoryPrototype_BatchTest/`를 만들었다.
- 복제본에서 `TinyFactoryMilestone16PlaytestRunner`를 배치 모드로 실행했다.
- 목표 장비는 `body_work_apron`으로 고정했다.
