# 마일스톤 19 배치 검증 로그

- `Body`, `Tool`, `Head` 공통 장비 3종의 실제 효과가 런타임 수치로 반영되는지 확인한다.

## 검증 방식

- 복제본 `unity/TinyFactoryPrototype_BatchTest/`에서 배치 플레이를 실행했다.

- 러너는 이동 속도, 완료 조립 시간, 출고 금액을 상태 파일로 기록했다.
