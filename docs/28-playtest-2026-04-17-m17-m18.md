# 마일스톤 17-18 배치 검증 로그

## 목적

- `Worker Gear` HUD의 장비 보너스/조합 가독성 문자열이 런타임에서 정상 계산되는지 확인한다.
- 스테이지 클리어 이후 추가 생산 보상 `Output 50 -> Basic Box +1`이 실제로 지급되는지 확인한다.

## 검증 방식

- 원본 Unity Editor는 유지했다.
- 복제본 `unity/TinyFactoryPrototype_BatchTest/`에서 배치 플레이를 실행했다.
- `TinyFactoryMilestone16PlaytestRunner`를 재사용하되, 상태 파일에 HUD용 문자열과 생산 보상 상태를 추가 기록하도록 보강했다.

- 생산 목표 문자열은 종료 시점에 `All production rewards cleared`로 바뀌었다.
- `Output 50` 달성 후 추가 `Basic Box`가 자동 개봉돼 `Hand Driver` 1개가 인벤토리에 들어왔다.
- 마지막 장비 메시지는 `Opened Basic Box: Hand Driver.`였다.
